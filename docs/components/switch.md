# Switch

Switches are a pleasant interface for toggling a value between two states, and offer the same semantics and keyboard navigation as native checkbox elements.

## Import

```javascript
import { VSwitch } from 'vuetify'
```

## Usage

<WrapView >
  <v-switch></v-switch>
</WrapView>

```html
  <v-switch></v-switch>
```
